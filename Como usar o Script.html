Como usar o script de Código de Ética

Importante: Você precisará da planilha enviada pelo time da UIC com os aceites e pessoas que concluíram o curso de Código de Ética.
  1. Renomeie a planilha:
    Antes de tudo, renomeie o arquivo que você recebeu para Arquivo Código de Ética.xlsx.
    Obs: O arquivo precisa obrigatoriamente estar no formato .xlsx (Excel).
  2. Substitua o arquivo na pasta correta:
    Coloque (ou substitua) o arquivo na pasta indicada.
  3. Execute o script de automação:
    Dê dois cliques no ícone do script para abrir.
    Em seguida, clique no botão Restart para iniciar.
  4. O que acontece em seguida?
    O script irá alimentar automaticamente o arquivo de relatório:
    relatorio_ja_enviados.xlsx com os novos dados processados.
  5. Baixe o arquivo final:
    O arquivo processado ficará disponível na pasta Arquivo para Envio com o nome
    Acessos UC - [data de hoje].xlsx.
    Para baixar, clique com o botão direito no arquivo e escolha Download.
Em caso de dúvidas, procure o suporte do time UAR.
